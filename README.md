# Graphyti – March 2026 Demo

## Overview
This repository hosts the materials and updates for the **March 2026 video demo** of **Graphyti**, our startup project focused on developing immersive 3D synthetic data for visual AI models.  
Graphyti is a platform that allows team to create 3D datasets faster, cheaper and more easily.

## Demo Video
▶️ **Watch the full demo:** [Graphyti Demo – March 2026](https://youtu.be/JAhphnwPNBY)

This demo presents:
- Procedural generation algorithm in real time  
- Intuitive camera motion, selection, and dynamic layout adjustment  
- Different camera modes including satellite
- Dataset capture & storage
- Full web user interface including AI design copilot
- AI video generation via Graphyti's spatial layout 

## Key Updates
- **Web UI:** Added new interactive UI over the web.  
- **UI Enhancements:** Added dataset capture and city configuration options.  
- **AI Design Copilot** AI assistant for designing full 3D city layouts and generating new videos.
